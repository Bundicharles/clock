# Glassmorphic Smartwatch PWA

A beautifully styled, Progressive Web App smartwatch clock and diary with offline persistence via IndexedDB, written in HTML, CSS, Vanilla JS, and built with Vite.

## Features
- **Glassmorphic UI**: Premium translucent styling with shiny tiled meshes.
- **Clock Faces**: Synchronized analog and digital clock views.
- **Alarms System**: Standalone Alarms with custom `.mp3` / `.wav` audio upload support that persist indefinitely offline.
- **Swipeable Diary**: A dedicated page imitating lined paper to save personal journal entries via a date picker.
- **Future Events**: Easily keep track of upcoming events aligned with their own toggleable bell alarms.
- **PWA Ready**: Cache-first Service Worker with offline compatibility and installable web manifest configuration.

## Setup
1. Clone the repository.
2. Run `npm install` to grab the dependencies.
3. Run `npm run dev` to start the local Vite server.
4. Run `npm run build` to generate the production bundle inside `dist/`.
